# 🔐 PASS CHECKER PRO

A modern Python-based Password Strength Checker and Password Generator built with **Tkinter**. This tool helps users analyze password strength, generate secure passwords, and view SHA-256 hash and Base64 encoding.

![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 📌 Features

- ✅ Password Strength Checker
- ✅ Secure Password Generator
- ✅ Password Rule Validation
- ✅ Password Length Detection
- ✅ SHA-256 Hash Generator
- ✅ Base64 Encoder
- ✅ Copy Password to Clipboard
- ✅ Show / Hide Password
- ✅ Clean Dark Hacker Style Interface

## 🔑 Password Rules

A strong password should contain:

- Minimum 8 characters
- At least one Uppercase letter
- At least one Lowercase letter
- At least one Number
- At least one Special Character

Example:

```
Cyber@Lee07lawat
```

---

## 🔐 Password Generator

The built-in password generator creates passwords that always include:

- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters

This ensures generated passwords satisfy common password complexity requirements.

---

## 🔍 Password Analysis

The application displays:

- Password Strength
- Password Length
- Estimated Crack Time
- SHA-256 Hash
- Base64 Encoded Output
